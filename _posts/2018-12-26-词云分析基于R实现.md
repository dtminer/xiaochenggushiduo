#### 需要的程序包

jiebaR，jiebaRD：分词

wordcloud2：绘制词云

```
installpackage('jiebaR','jiebaRD','wordcloud2) #安装程序包
```

#### 运行程序包

```r
library(jiebaR,jiebaRD) 
library(wordcloud2)
```

#### 导入数据

```r
text<-news$x #提取文本数据所在列
```

#### 分词

```r
mixseg<-worker("mix") #建立模型分词
```

```r
a<-segment(text,mixseg) #开始分词
```

#### 绘制词云

```r
a<-table(a) #词频统计
a   #查看词频统计结果
```
```r
wordcloud2(a,shape='star') #绘制词云
```
